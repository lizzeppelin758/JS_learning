№1. В следующем коде ключ должен был взяться из переменной. Исправьте допущенную ошибку: let key = 'x'; let obj = { key: 1, y: 2, z: 3 };

№2. Дан объект: let obj = { x: 1, y: 2, z: 3 }; Даны переменные: let key1 = 'x'; let key2 = 'y'; let key3 = 'z'; Сделайте так, чтобы ключи объекта брались из этих переменных.
