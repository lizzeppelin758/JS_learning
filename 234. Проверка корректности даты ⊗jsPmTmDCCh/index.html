1. Сделайте функцию checkDate, которая будет выполнять описанную проверку. Пусть функция возвращает true, если дата корректна и false, если нет. Пример работы этой функции для 31 января и 32 января: console.log(checkDate(2025, 0, 31)); // выведет true console.log(checkDate(2025, 0, 32)); // выведет false
