27. Константы. Создаю константу со значением Пи и вычисляю длину окружности с радиусом 15 см.

28. Автоматическое преобразование типов. Если в строках только числа, то при осуществлении арифметических операций над ними JS видит их как числовой тип переменной. Только сложение строк происходит как расположение текста рядом друг с другом. Если в выражении есть и строка, и числовая переменная, язык работает с ними всеми, как со строками.

29. Преобразование к числу. Функция Number(здесь строковая переменная, которую надо преобразовать в числовую). Слово Number можно заменить на + перед аписью строки. Это не всегда хорошо читаемо, в выражениях сложения лучше не использовать.

30. Выделение чисел из строки. Функция parseInt(здесь строка, состоящая из цифр и букв, из которой нужно взять только число). parseInt ивлекает только целую часть числа. Для взятия числа с дробной частью используется parseFloat.

31. Преобразование чисел к строке. Функция String(числовая переменная). Если преобразовать число к строке, то можно узнать число цифр в нем с помощью свойства length. вот так: String(число или идентификатор переменной числового типа).length

32. Преобразование логического типа (true и false) к строке или числу. преобразование к строке - получаются слова. преобразование к числу - получаются 0 и 1.

33. Преобразование к логическому типу. Все, что обозначает нуль, пустоту и несоответствие, преобразуется в false: 0, -0, +0, null, false, NaN, undefined, ''(это пустая строка). Всё остальное становится true, в том числе обе бесконечности, а также 0, -0, +0, null, false, NaN, undefined, когда они записаны в строчной переменной (они теряют свое смысловое значение и становятся словами или символами). Функция Boolean().

34. Получение символов строки по их номерам. Строка представляется массивом, каждый символ которого имеет номер. Нумерация начинается с 0. Отдельный символ строки можно получить с помощью операции: имя переменной[номер символа]. Изменить символ в уже существующей строке с помощью этой операции нельзя.

35. Вывести последний символ строки через свойство length. Найти длину строки, отнять 1 - это будет номер последнего символа в строке.

36. Строки с цифрами. Числовая сумма символов находится с помощью функции Number, примененной к каждому нужному символу.

37. Обращение к цифрам числа. Чтобы взять только некоторые цифры числа, нужно преобразовать его в строку функцией String и взять символы с нужными номерами. Если далее нужно выполнять арифметическую операцию сложения, преобразовать символы снова в числа функцией Number (иначе символы сложатся как строки и получится снова строка с теми же символами рядышком). Для всех остальных арифметических операций преобразование необязательно, язык сам делает его.
